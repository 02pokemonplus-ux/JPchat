操作說明

1

新增一份批改單

在 Claude 的日文聊天中輸入「結束對話」，會生成一份批改單 HTML，以及一段對應的 JSON 紀錄。把 HTML 存檔，檔名建議用日期，例如 2026-07-05.html。

2

上傳 HTML 到 GitHub

進入本 repo →「Add file」→「Upload files」→ 選剛才那份 HTML，放在跟 index.html 同一層 → Commit。

3

把紀錄加進 list.json（重點）

list.json 全程只有一個檔案，每次是「編輯」它、在裡面加一筆，不是上傳新的 json。

操作：點開 list.json → 右上鉛筆圖示 Edit → 把 Claude 給的那筆物件貼進 [ ] 陣列裡 → Commit。

格式提醒 新的一筆要放在最外層中括號 [ ] 之內，且與前一筆之間用逗號 , 隔開。最後一筆結尾不要加逗號。

一筆的長相：

{
  "date": "2026-07-05",
  "file": "2026-07-05.html",
  "topic": "聊梅雨季與騎車通勤",
  "topicJp": "梅雨と通勤の話",
  "count": 3,
  "tags": ["joshi", "goi"]
}

tags 可用值：joshi 助詞 · goi 語彙 · jitai 時態 · kanji 漢字 · other 其他（可複選）。

4

完成

重整本頁（Pages 網址），新卡片就會自動出現，點進去即可看批改單。首頁最上方的統計數字也會自動更新。

💡 看到的是原始碼而不是漂亮畫面？請確認你是從 GitHub Pages 網址（你的帳號.github.io/…）打開，而不是直接在 GitHub 檔案列表點 HTML。
